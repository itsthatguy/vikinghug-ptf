# VikinghugUI WOW Addon

An EXTREMELY opinionated player & target frame addon for WOW Classic.

## Current Features
- Player Health Bar
- Player Power Bar
- Player Cast Bar
- Player MP5 Bar
- Target Health Bar
- Target Power Bar
- Target Cast Bar
- Target Class Icons
- Supports Druid forms
- Masque Support
- LibSharedMedia Support (adds fonts and textures)

## In Progress Features
- Player Buffs
- Player Debuffs
- Player Weapon Enchants
- Player Shaman Totem Timers
- Target Buffs
- Target Debuffs
- Target MP5 Bar
- Map Coordinates

## Will not be implemented... ever...
- Extensive customization... this addon is extremely opinionated. Use it, or don't.
